Welcome!

This is my Blog Assignment for Globetom, as requested this Application does the following:

  • Allows the user to post a new blog entry.
  • Allows the user to a view list of all his/her blog entries.
  • Allows the user to edit and/or delete whichever blog entry she/he desires.


How to use the app:

  • To add a new blog entry, on the sidebar click "New Post", both fields are required.
  • To View a list of all the entries, on the sidebar click "View Posts".
  • To Edit or Delete a specific post, in the thumbnail, click the Edit button.
    • To edit the post, click edit. Both fields are required again.
    • To delete the post, click delete.
